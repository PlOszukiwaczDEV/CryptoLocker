# PlOszukiwacz Cryptolocker

## How to run
1. Clone this repo
```
git clone https://github.com/PlOszukiwaczDEV/Cryptolocker.git
```
2. Cd
```
cd Cryptolocker
```
3. Install requirements
```
pip install -r requirements.txt
```
or
```
pip install cryptography
pip install tkinter
```
4. Run
